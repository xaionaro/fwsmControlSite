# Pre-install

Install: https://github.com/xaionaro-go/fwsmAPI

# Install

```sh
curl -sL https://deb.nodesource.com/setup_9.x | sudo -E bash -
sudo apt-get install -y nodejs

git clone https://github.com/xaionaro/fwsmWebControl
```

# Test-start

```sh
npm start
```

(try localhost:3000)

# Production build

```sh
yarn build
```
